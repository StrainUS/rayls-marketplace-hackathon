## What it does

React frontend for the Rayls NFT marketplace. Connects to smart contracts for minting and trading invoice NFTs on the Rayls ecosystem.

## Tech Stack

- React 19
- TypeScript
- WalletConnect
- Rayls Testnet (Chain ID: 7295799)

## Run
```bash
npm install
npm start
```
